# testSonarHealth

To test the email validation its only requiere:
- at least one number or letter 
- Follow by the character "@"
- at least one number or letter 
- and in the end a "." + at least two letters

Examples:
adsa@sonarhealt.co -> Valid
dasdasd@sonarhealt.c -> Invalid
